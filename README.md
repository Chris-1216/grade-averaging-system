# C++ Grade Averaging System

This is a simple C++ program that reads student grade data from a file, calculates each student's average, converts the average to a letter grade, and displays the results.

## Features
- Reads multiple students from an input file
- Computes average grade for each student
- Converts numeric averages to letter grades
- Displays each student's full name, average, and final letter grade
- Displays class average, highest average, and lowest average

## Files
- `main.cpp`
- `example-grades.txt`

## Input Format
The first number in the file is the total number of students.

Each student entry contains:
- first name
- last name
- number of grades
- that many grade values
